KMS 激活命令

2020-04-18
1 min read
kms

快速跳转: Windows | Office
备注： 需要管理员权限运行Powershell

Windows KMS

  • 查看系统版本
    
wmic os get caption
  • 设置kms服务器
    
slmgr /skms 192.168.0.1
  • 安装key | [ 需要Key? 点这里： Windows ]
    
slmgr /ipk xxxx-xxxx-xxxx-xxxx
  • 激活
    
slmgr /ato

Office KMS

  • 设置kms服务器
    
#32位
cd c:\Program Files (x86)\Microsoft Office\Office16\
#64位
cd c:\Program Files\Microsoft Office\Office16\
#运行
cscript ospp.vbs /sethst:192.168.0.1
  • 安装key | [ 需要Key? 点这里：Office2019/2016 | Office2013 | Office2010 ]
    
cscript ospp.vbs /inpkey:xxxxx-xxxxx-xxxxx-xxxxx
  • 激活
    
cscript ospp.vbs /act
